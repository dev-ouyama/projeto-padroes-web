## Sobre o projeto

O projeto **Globalização Digital** foi desenvolvido como atividade da disciplina **ICSM49 - Padrões Web** da UTFPR.

A proposta é estimular uma reflexão sobre como algoritmos, redes sociais e plataformas digitais influenciam nossos interesses culturais. Para isso, foi desenvolvido um site contendo um quiz interativo, materiais explicativos sobre globalização digital e referências para aprofundamento no tema.

O objetivo do quiz não é avaliar conhecimentos, mas incentivar o usuário a analisar seus próprios hábitos de consumo cultural e perceber como conteúdos nacionais e internacionais fazem parte de sua rotina.

## Como executar

Clone o repositório:

```bash
git clone https://github.com/dev-ouyama/projeto-padroes-web.git
```

Entre na pasta do projeto:

```bash
cd projeto-padroes-web
```

Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como **Live Server** no Visual Studio Code para executar o projeto.

## Funcionalidades

- Página inicial com apresentação do tema
- Quiz interativo
- Navegação entre perguntas
- Embaralhamento das questões e alternativas
- Exibição dinâmica de imagens nas perguntas
- Barra de navegação entre questões
- Modal de confirmação ao sair do quiz
- Página de resultados
- Página "Sobre" com explicações sobre o tema
- Layout responsivo para dispositivos móveis
- Links para artigos e materiais complementares
